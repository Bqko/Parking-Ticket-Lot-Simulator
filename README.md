# Parking Ticket Lot Simulator

Java/JavaFX desktop app for managing a parking lot ticket workflow:
vehicle entry, spot assignment, ticket payment, exit, dashboard metrics,
admin pricing, admin authentication, customer profiles, and SQLite persistence.

## Requirements

- JDK 21 or newer
- Maven 3.9 or newer

## Run Tests

```bash
mvn test
```

## Run The App

```bash
mvn javafx:run
```

## Default Admin Login

